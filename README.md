# Applyo 🚀

Applyo is a modern, full-stack job application platform designed to help users discover, save, and apply to jobs seamlessly while providing companies with a streamlined way to post opportunities and manage applications.

## ✨ Features

- **Job Discovery & Search:** Instantly search and filter jobs using **Algolia** integration for fast, relevant results.
- **User Authentication:** Secure, custom JWT-based authentication system with password hashing (`bcryptjs`).
- **Dashboard & Tracking:** Keep track of saved jobs, applied positions, and application statuses.
- **Company Profiles:** Employers can create company profiles, post new jobs, and review applications efficiently.
- **Modern UI/UX:** Built with **Tailwind CSS v4**, **Radix UI** primitives, and **Framer Motion** for a beautiful, responsive, and animated user experience.
- **Dark/Light Mode:** Full theming support respecting system preferences out of the box.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router, Server Components, Turbopack)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & [Radix UI](https://www.radix-ui.
